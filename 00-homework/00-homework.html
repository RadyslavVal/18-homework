Command Description
git config --global user.name yourName Встановити ім'я користувача
git config --global user.email blabal@bla Встановити адресу електронної пошти
git init Створює порожній репозиторій Git, або повторно ініціалізуйте наявний
git status Показує стан робочого дерева
git add Вибирає файли для commit
git commit Записує зміни до репозиторія
git log Показує журнал фіксації змін
git checkout Перемикає між гілками, або відновлює файли робочого дерева
git branch Можна створити, видалити чи вивести список гілок
git push Передає commit в віддалений репозиторій з локального
git pull Оновлює дані в локальному репозиторії з віддаленого
git merge Об'єднує декілька гілок разом
git clone Клонує репозиторій в нову папку
git revert Записує нові commit, що скасовують попередні commit
git reset Є протилежним до git add
git rebase Для переміщення послідовності commit
git cherry-pick Застосовує зміни декількох commit
git stash Дозволяє відкласти зміни в робочій копії до "архіву"
